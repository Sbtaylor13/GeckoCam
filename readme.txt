# GeckoCam

This is a webpage containing a livestream of Lemon the Leopard Gecko

## Live Stream Link
http://shanetaylor.info/

## Features 
- **Live Video Feed:** GeckoCam is available 24/7 under the GeckoCam tab on the home page
(coming soon)
- **Chat Interaction:** A chat is available for all users to talk and update each other on Lemon's whereabouts
- **Animal Fact Board** A fact board users can submit to and also like other user's posts
